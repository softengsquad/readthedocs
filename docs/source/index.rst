Welcome to Student Map's documentation!
===================================

Student Map is a university map app and timetable for students. It provides
university building information, such as location, navigation and room
locations. It will also display timetabled sessions from the user's Google 
calendar.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
